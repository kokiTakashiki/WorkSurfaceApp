# WorkSurfaceApp

# 概要
WorkSurfaceAppは個人学習を助けるアプリである。
作業机では作業に必要なツールを一時的に自由に広げることができる。
作業が終われば、そのツールをしまうことができる。
広い空間が新たな創造を掻き立てるのだ。

# 機能
本アプリは次の章で説明する学習ツールを起動させることができる。
本アプリは起動させたツールを全て終了させることができる。
本アプリは寄贈させたツールをユーザが指定したツールのみ終了させることができる。

## 学習支援ツール
### 学習支援
・ポモドーロタイマー
・画像メモ
・アプリショートカット
・Webアプリショートカット
・録音機能
・学習中の質問（ChatGPT）

### 学習振り返り支援
・学習の記録
・振り返りメモ
・振り返りFB（ChatGPT）

### 学習計画支援
・学習予定の計画
・目標設定
